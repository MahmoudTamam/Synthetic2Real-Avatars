Synthetic to Real Faces Dataset

A Benchmark for Domain Adaptation and Facial Recognition

Overview

The Synthetic to Real Faces Dataset provides high-resolution synthetic face images paired with real-world counterparts. This dataset enables research in domain adaptation, facial recognition, and synthetic-to-real generalization.

Features

  • High-quality synthetic faces generated using GANs
  • Real-world face images from public datasets
  • Pairwise alignment between synthetic and real images
  • Variations in lighting, pose, and expression
